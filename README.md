# Rock Paper Scissors Game

An interactive Rock-Paper-Scissors game with an autoplay feature.

## Features
- Play against the computer.
- Autoplay mode that continuously plays automatically.
- Score tracking.

## Tech Stack
- HTML5
- CSS3
- JavaScript
